Introduction to Interblockchain Communication and the Cosmos SDK

What is Interblockchain Communication?

Today there are thousands of networks, permissioned and permissionless. These networks work great on their own, but to realize their true potential, we need communication between Blockchains. From Atomic Swaps to Shared Contracts, there is a lot of potential. Interblockchain Communication are just a set of protocols that allow for communication between Blockchain networks.

Cosmos is a decentralized network of independent parallel blockchains, each powered by BFT consensus algorithms like Tendermint consensus.

Starport is a tool to help you create Blockchains using the Cosmos SDK.

Set up Starport

git clone https://github.com/tendermint/starport && cd starport && make

Then move Starport binaries to $PATH or add the binary directory to environment variable. Make sure the $GOPATH is correctly set. Most problems are solved with a fresh install or checking if the variables are set right. Also ensure your Golang is at the most updated stable version. Currently I used Go1.15.8 with Starport v0.13.2-52-g7f8bd92.

Adding the following to .bashrc solved most problems I had.

export GO_PATH=~/go
export PATH=$PATH:/$GO_PATH/bin
export PATH=$PATH:$(go env GOPATH)/bin

Install Relayer

Relayer is an off-chain protocol used to relay packets between IBC compatible chains (eg. Gaia and IRIS). You might have to build it locally for Stargate update. Please make sure you relayer versions match with Starport’s requirements.

git clone https://github.com/cosmos/relayer && make install

(Optional) Use the Web IDE with preinstalled binaries

If you’re building a Proof-of-Concept, you might want to use the Gitpod Environment. Starport and Relayer come installed in this environment.

(Optional) Using Docker to set up your environment

You can pull the Gitpod workspace as a Docker image. I have personally not tried this but this should work.

docker pull tendermintdevelopment/starport
